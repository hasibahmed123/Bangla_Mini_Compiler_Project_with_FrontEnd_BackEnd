পরিক্ষা ১: Arithmatic

 লেখ নয় গুণ সাত; 
লেখ পাঁচ গুণ দুই বিয়োগ তিন;
Parentheses
লেখ ( এক যোগ দুই ) গুণ তিন;

string:
লেখ "বাংলা কম্পাইলার কাজ করছে";

পরীক্ষা 2: যদি-নাহলে
ইনপুট: যদি (দুই > এক) { লেখ "সঠিক"; } নাহলে { লেখ "ভুল"; }

পরীক্ষা 3: যতক্ষণ লুপ
ইনপুট: a = ১; যতক্ষণ (a < ৪) { লেখ a; a = a + ১; }

পরীক্ষা 4: প্রতিবার লুপ
ইনপুট: প্রতিবার (k = ০; k < ৫; k = k + ২) { লেখ k; }
